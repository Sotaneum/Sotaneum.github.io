## Command

| 구분 | 설명                                   |
|--|--------------------------------------|
|npm run new| 새로운 글을 작성할 때 사용합니다. 기본 포맷을 맞춰 생성합니다. |
|npm run last| 마지막 게시글을 불러옵니다.                      |
|npm run compile| 이미지를 assets에 저장합니다. (빌드/dev 시 자동 실행) |
